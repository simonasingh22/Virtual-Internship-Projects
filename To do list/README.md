# MERN To-Do App

This workspace contains a full-stack MERN To-Do application example.

Folders:
- `server` — Express + Mongoose API
- `client` — React + Vite + Tailwind frontend

Follow README files in each folder to run locally.

## Quick start

1. Backend

```bash
cd server
cp .env.example .env
npm install
npm run dev
```

2. Frontend

```bash
cd client
cp .env.example .env
npm install
npm run dev
```

API testing: import `server/postman_collection.json` into Postman and set `{{baseUrl}}` to `http://localhost:5000`.
